# Mailman Sandbox

`mailman` 스킬의 호환 별칭. 동작은 동일하다.

```bash
bash ~/plugins/mailman/runtime/fetch.sh [스페이스별칭] [봇별칭] [개수]
```

자세한 사용법은 `mailman` 스킬 문서 참고.
